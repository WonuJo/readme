# RMS FRONTEND

## 설명
기기 관리, 고객사 별 배송 관제 확인, C/S 처리 등 내부에서 사용하기 위한 Web 입니다.

## 기술스택
```React, Apollo, GraphQL, TypeScript, Recoil```

## 설치 및 실행
종속성 설치를 위해 아래와 같이 입력합니다.
```
yarn
```

실행을 위해 아래와 같이 입력합니다.
```
yarn start
```

배포 시, 아래와 같이 입력합니다.
```
yarn build
```

## ETC..
digital envelope routines 등의 에러 발생 시 node version 을 낮춰 실행합니다.
