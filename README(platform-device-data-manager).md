# PLATFORM DEVICE DATA MANAGER BACKEND

## 설명
Platform 의 Web 과 App (fresh-monitor 등..) 의 Core Back-End 입니다.

## 기술스택
```
언어: TypeScript
프레임워크: NestJS
데이터베이스: InfluxDB
웹 프레임워크: Express
테스팅 프레임워크: Jest
HTTP 클라이언트: axios
로깅: Winston
환경 변수 관리: dotenv
API 문서화: Swagger UI Express
빌드 도구: ts-loader
패키지 매니저: Yarn
```

## 설치 및 실행
종속성 설치를 위해 아래와 같이 입력합니다.
```
yarn
```

실행을 위해 아래와 같이 입력합니다.
```
yarn start
```
