# PLATFORM MOBILE WEB

## 설명
Platform 의 Monbile 환경의 Web Front-End 입니다.

## 기술스택
```React, TypeScript, apollo, recoil```

## 설치 및 실행
npm 등 package 설치를 위해 아래와 같이 입력합니다.
```
yarn
```

실행을 위해 아래와 같이 입력합니다.
```
yarn start
```

배포 시, 아래와 같이 입력합니다.
```
yarn build
```
