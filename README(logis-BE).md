# LOGIS BACKEND

## 설명
LOGIS 의 Web 과 App (Logis scanner, Logis linker 등..) 의 Back-End 입니다.

## 기술스택
```NestJS, TypeScript, GraphQL, MySQL, TypeORM```

## 설치 및 실행
종속성 설치를 위해 아래와 같이 입력합니다.
```
yarn
```

실행을 위해 아래와 같이 입력합니다.
```
yarn start
```

## ETC..
digital envelope routines 등의 에러 발생 시 node version 을 낮춰 실행합니다.
